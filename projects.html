Distributed Computing Using Web Workers

The goal of this project is to showcase the use of the Web Worker API for distributed computing.

The current demo searches for the amount of prime numbers in the range 0-400 by giving each worker a range of 100 digits. In theory, this is not the most efficient way to implement distributed computing for prime numbers, but this is just a demonstration on how multiple web workers can be used to compute one big task.
