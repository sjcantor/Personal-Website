QMusic

*In Development*
This is my Music project from QMIND 2019. It is a machine learning application that uses an LSTM neural network to generate original music based off any input image. It was trained using data from the Spotify API, and each 30 second audio clip was matched up with it's corresponding cover artwork. The neural net would predict what the audio should be for any given input image, and slowly got better at making predictions and finding patterns. This project was developed in python, but I am currently converting the model to tensorflow js, so that it can be hosted with only JavaScript. For now, you can check out the GitHub repo for our project.
